## 다음 작업 시 확인할 것 (최우선)

### 블록 attribute 체크 목록
AutoCAD에서 각 블록 편집(BEDIT)으로 진입해 확인:

**M_PKA0102 (펌프)**
- [ ] `IN1` attribute 존재 + 위치 확인 (좌측 바깥이어야 함)
- [ ] `IN1_ANG` attribute 존재 여부 + 값 (180이어야 함)
- [ ] `OUT1` attribute 존재 + 위치 확인
- [ ] `OUT1_ANG` attribute 존재 여부 + 값

**FIT_FLNG (플랜지)**
- [ ] `IN1` attribute 위치 = 블록 IN쪽 끝단
- [ ] `OUT1` attribute 위치 = 블록 OUT쪽 끝단 (IN1과 달라야 체인 전진 가능)
- [ ] `IN1_ANG`, `OUT1_ANG` 존재 여부

**S_COND01 (구조물)**
- [ ] `IN1_ANG` 값 확인 (90이어야 함, 현재 0으로 읽힘)

### 확인 결과에 따른 다음 작업
- **attribute 없음**: 블록에 추가 or LISP에 code_key별 각도 하드코딩 테이블 추가
- **attribute 있는데 값이 0**: 블록 내 값 수정 (BEDIT → 해당 속성 → value 수정)
- **attribute + 값 정상**: LISP의 `get-attr` 함수 읽기 로직 문제 → 코드 재점검
